为了保证代码始终是最新的，您可以遵循以下几个基本步骤：

1. 开始工作前，先拉取最新代码： 在您开始编写代码之前，可以通过终端进入 d:\blog\portfolio 文件夹，执行以下命令，把另一台电脑上提交的更新抓取下来：

powershell
cd d:\blog\portfolio
git pull
2. 结束工作前，提交并推送到 GitHub： 每天或者每次完成部分工作后，执行以下命令将您的修改上传到 GitHub，这样另一台电脑就能看到：

powershell
# 将所有修改添加到暂存区
git add .
# 提交这些修改（请在双引号内填写您这次修改了什么内容）
git commit -m "更新您的修改说明，比如：完成了首页的设计"
# 推送到 GitHub
git push
