# 博客/作品集 V2 开发设计文档 (Game Advertising Designer 版)

> **设计背景**：基于 Shiro (Yohaku) 视觉风格，为游戏广告设计师打造的个人展示空间。强调视觉冲击力、阅读体验以及对广告投放作品的深度展示。

## 1. 核心目标
- 建立一个符合设计师身份的 V2 站点，采用 Next.js 16 和 Strapi 5。
- 实现“上升粒子”背景效果与“波浪”加载动画。
- 优化作品集展示，支持 30s 以内广告短视频的自动预览，并展示投放核心数据。
- 建立清晰的导航结构：首页、简历、作品、手记。

## 2. 视觉风格与 UI 系统
- **背景**：深色背景 + 上升粒子 (CodePen 效果)。
- **加载态**：全屏波浪动画 (CodePen 效果)。
- **动效**：Framer Motion 驱动的平滑切换；去除了原有的二级联动导航，保持导航栏简约。
- **排版**：中英文混排优化，手记页面保持极简阅读体验，风格与全站统一。

## 3. 页面结构与导航
- **导航 (Navbar)**：首页 → 简历 → 作品 → 手记 (无二级联动)。
- **首页 (Home)**：
  - **首屏**：Slogan + 简短个人介绍。
  - **内容区**：
    - 精选视频作品（2 排，每排 2-3 个卡片，带自动预览）。
    - 精选图片作品（2 排，每排 3-4 个卡片）。
- **简历 (Resume)**：个人简介、品牌墙、工具链、经历。
- **作品 (Works)**：
  - 支持类型筛选（视频/图片）。
  - **数据展示**：每项作品需标注“花费 (Spend)”和“7日付费ROI”。
- **手记 (Blog)**：广告创意文章分享，保持 Shiro 简约风。

## 4. 技术与部署方案
- **前端**：Next.js 16 (Vercel 部署)。
- **后端**：Strapi 5 (本地开发 + 云端部署)。
- **数据库**：Neon (PostgreSQL)。
- **存储**：Cloudflare R2 (用于存储作品视频和图片)。
- **CDN/DNS**：Cloudflare。

## 5. 详细数据模型 (Strapi)
- **Works Collection**:
  - `Title`: 标题
  - `Type`: 枚举 (Video, Image)
  - `Cover`: 封面图
  - `VideoFile`: 视频文件 (R2)
  - `Spend`: 投放花费
  - `ROI_7D`: 7日付费ROI
  - `Description`: 创意说明
  - `IsFeatured`: 是否首页精选
- **Articles Collection**:
  - `Title`: 标题
  - `Content`: 内容 (Markdown)
  - `Tags`: 标签
